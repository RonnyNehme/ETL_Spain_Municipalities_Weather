# ETL_Spain_Municipalities_Weather

Python web scraping of all the municipalities in Spain. (https://es.wikipedia.org/wiki/Anexo:Municipios_de_Espa%C3%B1a_por_poblaci%C3%B3n)
Uploading each municipality as a file to google cloud.
Creating a SQL database of municipalities.
Finding the top 15 most populous municipalities from API.
Weather prevision for those 25 municipalities. (https://api.open-meteo.com/v1/forecast/")
